sorry, 上周因为看导致进度没有如期而至.现在简单补上.

hexo

  • instantclick
    无刷新页面快速载入,当鼠标悬停在链接上的时候,就开始加载过程,从而极大加快页面的载入速度.
  • Table of Content
    加上目录功能,目录的CSS样式参考了维基百科和light主题.
  • 前向后向页面
    在文章的最后增加前向后向的页面链接功能

这个后来发现又有bug,临时撤掉了.

  • 百度统计
    话说增加了百度统计,结果就有的百度统计被利用攻击github的事情.
    现在正在考虑是否去掉这个统计功能.

gerrit

这部分见gerrit相关的文章,目前还没有完成.

linenoise

发现一个linenoise的Line Feed和Carriage Return,也就是”\r”和”\n”的bug.

这个问题就是在Android的adb的shell环境下,得到的回车键的响应符号是”\r”.而linenoise没有考虑这个情况.

这个目前已经反映到其在github的页面上.不过那个似乎没有人维护,也就放在那里吧,有需求的人会看到的.

git-review

发现一个不能处理UTF8字符的bug.也已经处理了.

但是其bug反馈的流程需要经过openstack的组织的review,现在还没有正式提交.

其实这个bug也比较无所谓,就先这样吧.

web application developing

另外mongodb确实比较适合”轻量级”程序员啊.
