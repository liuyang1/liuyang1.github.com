上海浦东图书馆

本周来了两次浦东图书馆.这里不错.

  • 敞亮,绿色,开放式的环境
  • 每个座位下面还设有电源(USTC多年前就有了,这点很不错)
  • 座位比较多,不过上午十点以后,估计就没有多少啦.

缺点:

  • 无线网的设置和覆盖
    信道的设置,共享SSID,稳定性等还有很多要增强. 不过考虑到想学习,这个反而可能是一件好事.

推荐有读书爱好的,可以来这里感受一下.

这篇博客就是在浦东图书馆诞生的.

BitBucket

我其实一直把自己的代码托管在github上面,现在因为coursera的web application课程的缘故,也顺便接触了下BitBucket.

这二者确实有很多不一样的地方.侧重的点确实很不同,自己多了解以后,再在这个话题上展开吧.

Web Application @ Coursera

最开始接触这个主要是觉得自己关于前端和web方面了解的实在太少了.

说实话,这个课程觉得不是很推荐的感觉.

这个课程是以Ruby on Rails为基础介绍相关的内容,虽然RoR已经没有那么In了,其实我是无所谓的.
但是里面有些细节总是没有说清楚的样子.

目前阶段上来看,难度不大.如果可以更清晰一点就更好啦.
